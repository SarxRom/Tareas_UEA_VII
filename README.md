También se mencionan las carpetas de tareas que se han entregado, dentro de estas carpetas están colocados los archivos de index ya que como todo este repositorio se realizan pruebas en lenguaje de html es necesario poner estos archivos en carpetas para que estas se pueden desplegar sin necesidad de que se mezclen entre sí.  
  
## Experiencia personal de crear y gestionar tu repositorio.  
Mi experiencia al gestionar y crear mi repositorio ha sido muy gratificante, al pasar de lo trimestres he podido entender más el uso de **_GIT HUB_** y el uso de **_VISUAL STUDIO CODE_** ya que en este software en donde podemos programar y subir desde la terminal.  
  
Creo que el usar estas tecnologías nos ayudan a mantener más organizado nuestros proyectos y nos ayudan a almacenar nuestros ejemplos para poder utilizarlos si es necesario, aparte que si necesitamos trabajar en colaboración es en lo personal mas facil ya que todos pueden tener el codigo y poder modificar sin la necesidad de ocupar todos una misma laptop.  
  
Aparte esto nos ayuda a tener más organización y comunicación con la personas que trabajamos, para no modificar los espacios que uno ya está modificando y así realizar de una manera más eficiente el trabajo.
